/*Define the terms: Website, Webpage, Web browser, Web server, HTML, CSS
Website: A collection of related web pages, including multimedia content, identified by a common domain name and published on at least one web server. It's a digital presence accessible via the internet.
Webpage: A single document, typically written in HTML, that is viewable in a web browser. It's one specific "page" within a website.
Web browser: Software that allows users to access, retrieve, and view information on the World Wide Web. Examples include Chrome, Firefox, Safari, Edge.
Web server: A computer program that stores website content (like HTML documents, images, videos) and delivers it to web browsers upon request. It's the "host" for websites.
HTML (HyperText Markup Language): The standard markup language for creating web pages and web applications. It provides the structure of the content (headings, paragraphs, links, images).
CSS (Cascading Style Sheets): A style sheet language used for describing the presentation of a document written in HTML (or XML). It controls the layout, colors, fonts, and overall appearance of web pages.*/
